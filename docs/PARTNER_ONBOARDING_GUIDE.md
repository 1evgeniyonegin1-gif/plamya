**Версия:** 1.0
**Дата:** 21 января 2026

## Шаг 1: Создание Telegram-бота

### 1.1 Открой @BotFather

1. В Telegram найди бота **@BotFather**
2. Нажми **Start** или отправь `/start`

### 1.2 Создай нового бота

1. Отправь команду: `/newbot`

2. BotFather спросит **имя бота** — это то, что видят пользователи.
   Напиши, например:
   ```
   Анна | NL Куратор
   ```
   или
   ```
   Миша - Здоровье и Бизнес
   ```

3. BotFather спросит **username бота** — это техническое имя, должно заканчиваться на `bot`.
   Напиши, например:
   ```
   anna_nl_curator_bot
   ```
   или
   ```
   misha_health_nl_bot
   ```

4. BotFather пришлёт **токен** — это длинная строка вида:
   ```
   7123456789:AAHx1234567890abcdefghijklmnop
   ```

   **ВАЖНО:** Сохрани этот токен! Никому не показывай, кроме администратора системы.

### 1.3 Настрой описание бота (опционально)

Отправь BotFather:
```
/setdescription
```

Выбери своего бота и напиши описание, например:
```
Привет! Я AI-помощник Анны по здоровью и бизнесу NL International.

Помогу подобрать продукты под твои цели и отвечу на вопросы.
```

### 1.4 Установи аватарку бота (опционально)

Отправь BotFather:
```
/setuserpic
```

Выбери своего бота и отправь фото (твоё фото или логотип).

## Шаг 2: Создание канала в Telegram

### 2.1 Создай канал

1. В Telegram нажми **меню** (три полоски) → **Новый канал**
2. Назови канал, например: `NL | Анна`
3. Добавь описание (опционально)
4. Выбери **Публичный** или **Приватный** — как хочешь

### 2.2 Добавь бота в канал

1. Открой канал → **Настройки** (три точки вверху)
2. **Администраторы** → **Добавить администратора**
3. Найди своего бота по username (например: @anna_nl_curator_bot)
4. Включи права:
   - Публикация сообщений
   - Редактирование сообщений
   - Удаление сообщений

### 2.3 Получи ID канала

**Способ 1: Через бота @getidsbot**
1. Добавь @getidsbot в канал как администратора
2. Он напишет ID канала (начинается с `-100...`)
3. Удали бота из канала после этого

**Способ 2: Через веб-версию**
1. Открой web.telegram.org
2. Зайди в канал
3. В адресной строке будет: `web.telegram.org/a/#-1001234567890`
4. ID канала = `-1001234567890`

## Шаг 3: Собери ссылки из NL

### 3.1 Войди в личный кабинет NL

1. Зайди на https://nlstar.com/ (или твой региональный сайт)
2. Авторизуйся

### 3.2 Найди свой ID партнёра

Обычно отображается в личном кабинете или в URL.
```
Пример: 12345678
```

### 3.3 Найди реферальную ссылку на магазин

1. В кабинете найди раздел **Реферальные ссылки** или **Пригласить**
2. Скопируй ссылку на магазин с твоим ID:
```
Пример: https://nlstar.com/shop/?ref=abc123
```

### 3.4 Найди ссылку на регистрацию партнёра

```
Пример: https://nlstar.com/ref/abc123/
```

## Шаг 4: Заполни анкету

Заполни Google-форму (ссылка от администратора) или отправь информацию напрямую:

### Данные для анкеты:

**1. Основное:**
- Твоё имя (как бот будет представляться)
- Телеграм для связи с клиентами (@username)
- Телефон (опционально)

**2. Telegram-бот:**
- Токен бота (из BotFather)
- Username бота (@...)

**3. Telegram-канал:**
- ID канала (начинается с -100...)
- Название канала или @username

**4. Ссылки NL:**
- ID партнёра NL
- Ссылка на магазин
- Ссылка на регистрацию партнёра

**5. Профиль бота:**
- Специализация (выбери 1-2):
  - Похудение / контроль веса
  - Энергия / витамины
  - Бизнес / заработок
  - Красота / уход
  - Всё понемногу

- Твоя история (2-3 предложения):
  Почему ты в NL? Какие результаты у тебя или твоих клиентов?

- Стиль общения:
  - Дружелюбный, на "ты" (рекомендуется)
  - Более формальный, на "вы"

## Что будет дальше

1. **Администратор получает твою анкету** и проверяет данные
2. **Настраивает бота** на сервере (обычно в течение 24 часов)
3. **Отправляет тебе подтверждение** — бот готов к работе
4. **Проводится тест** — проверяем, что всё работает

## Частые вопросы

### Я не технарь, справлюсь?

Да! Инструкция пошаговая, занимает около 1 часа. Если что-то непонятно — пиши, поможем.

### Что если я что-то сделаю не так?

Ничего страшного. Все данные проверяются перед запуском. Если что-то не так — администратор напишет, что нужно исправить.

### Могу ли я изменить настройки потом?

Да, можно изменить: имя бота, специализацию, историю, ссылки. Просто напиши администратору.

### Сколько клиентов может обрабатывать бот?

Практически неограниченно. Бот работает на мощном сервере.

### А если Claude API заблокируют?

Используется общий аккаунт администратора с резервным провайдером (YandexGPT). Боты продолжат работать.

### Как следить за статистикой?

В боте будут команды:
- `/my_stats` — твоя статистика (лиды, конверсии)
- `/my_leads` — список твоих лидов
